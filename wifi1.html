How to Find WiFi Password in CMD with Netsh WLAN Show Profiles Command

If you’ve ever connected to a WiFi network and entered the password of the network, it will create a WLAN profile of that network stored in your computer. This WLAN profile contains some details of the WiFi network. You can use Netsh WLAN show password commands to easily find WiFi password of any available network in your Windows 10 computer. Check how to do it below.

1. Open elevated Command Prompt. You can press start, type cmd, and press Enter to run Command Prompt.

2. Show all stored available network profiles. Next you can type this command netsh wlan show profile in CMD window, and hit Enter. This command can list all the WiFi network profiles that you’ve ever connected. Record the WiFi network names.

3. Next you can type the command below to use netsh wlan show profile to show password list in CMD. Hit Enter after you type the command.

4. Next you can type the comman netsh wlan show profile name="wifiname" key=clear (replace wifiname with the actual WiFi network name which you want to find the password.

5. Password would be in the Key Content bar

Tip: If you want to connect a particular WiFi network you can type the command netsh wlan connect name = WiFi Name. To disconnect that WiFi, you can type netsh wlan disconnect name = WiFi Name. To save the network profile information, you can type the command netsh wlan export profile name = WiFi Name, and hit Enter. You should replace “WiFi Name” with the real network name of the target WiFi your computer detected.
